LRFA.dev

A collection of tools and projects built by Lachlan Alston.

BenchCheck Fast all-in-one hardware test. Check speakers, microphones, and webcams safely in a single browser screen — no data collected. EdgeScope Inspect and debug browser edges, layouts, and viewport behavior in real time. Tools Browse and access scripts, commands, and bookmarks via a clean web interface. Tracely A diagnostic tool built for technicians to quickly identify and troubleshoot Internet and NBN issues. Atlas Everything you need for IP-based threat investigation and helpdesk diagnostics. SecuRadar Threat intelligence tool that connects to ASD & CISA KEV to surface active vulnerabilities and known exploited CVEs. Eventful Event management and tracking tool. BrowserLab Browser testing and compatibility tool.
